# 蓮境：周敦頤的最後考驗

一款以〈愛蓮說〉為核心的沉浸式古風密室逃脫閱讀素養遊戲（國中二年級）。
HTML5 + CSS3 + JavaScript ES6，無大型框架。

## 如何遊玩

本遊戲使用 ES Module，**不能直接雙擊 index.html 開啟**（瀏覽器 CORS 限制）。
請在 `蓮境/` 資料夾啟動一個本機伺服器：

```bash
# 方法一：Python（多數電腦內建）
cd 蓮境
python3 -m http.server 8000
# 然後瀏覽器開啟 http://localhost:8000

# 方法二：Node
npx serve 蓮境
```

## 遊戲流程
開場 → 世界觀 → 教學（掃葉人）→ 五關試煉 → Final Ending →（隱藏結局）→ 結算

| 關卡 | 院落 | 閱讀素養 | 核心句 |
|---|---|---|---|
| 一 | 淤泥門 | 資訊擷取 | 出淤泥而不染 |
| 二 | 三花圃 | 比較分析 | 濯清漣而不妖 |
| 三 | 通直廊 | 推論判斷 | 中通外直，不蔓不枝 |
| 四 | 濂溪堂 | 作者觀點 | 香遠益清…褻玩焉 |
| 五 | 蓮心堂 | 統整反思 | 噫！…同予者何人 |

## 操作
- 點擊場景中「✦」探索彩蛋（16 個文化知識點）
- 「開始試煉」進入解謎（墨痕顯影／語序拼接／象徵配對／對比篩選／環境機關）
- 右下「燈籠」= 三階提示；閒置 3 分鐘 NPC 自動提示
- 左下「竹編包」= 查看已得殘句
- 進度自動存於瀏覽器 LocalStorage，重新整理可續玩
